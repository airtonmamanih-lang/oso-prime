Título del Proyecto de Tesis / Investigación

Optimización de Redes Neuronales Convolucionales (CNN) para la detección temprana de anomalías radiológicas en el sector de salud pública del Perú.

Resumen / Abstract

El presente trabajo desarrollado en la FISI - UNMSM propone un modelo optimizado basado en la arquitectura ResNet-50 para el procesamiento de imágenes de rayos X de tórax. El objetivo principal consiste en reducir los falsos negativos en el diagnóstico de patologías pulmonares prevalentes en zonas vulnerables de Lima Metropolitana. Mediante técnicas de transferencia de conocimiento (Transfer Learning) y aumentación de datos adaptativa, se logró un entorno eficiente capaz de ejecutarse en hardware de recursos limitados, facilitando su futura implementación en centros de salud periféricos.

1. Planteamiento del Problema Tecnológico

En los centros de salud asistenciales del Perú, la disponibilidad de médicos especialistas en radiología es severamente limitada. El procesamiento digital de imágenes asistido por computadora (CAD) surge como una alternativa viable. No obstante, los modelos de Deep Learning convencionales exigen capacidades de cómputo (GPUs de alta gama) inaccesibles para la realidad de muchas postas médicas locales.

Por lo tanto, la investigación de la FISI responde a la pregunta: ¿Cómo reestructurar hiperparámetros y capas densas para disminuir la carga computacional en FLOPS sin degradar la precisión del modelo en diagnósticos críticos?

2. Arquitectura de Software y Algoritmo

Se implementó un pipeline en Python utilizando la librería PyTorch. A continuación, se detalla un segmento modular del script diseñado en nuestros laboratorios para la congelación selectiva de capas (Layer Freezing): 

import torch
import torchvision.models as models

def inicializar_modelo_fisi(num_clases=3):
    # Carga de ResNet preentrenada
    modelo = models.resnet50(pretrained=True)
    
    # Congelar las primeras 4 capas residuales para preservar características base
    contador = 0
    for child in modelo.children():
        contador += 1
        if contador < 7:
            for param in child.parameters():
                param.requires_grad = False
                
    # Reestructuración de la capa totalmente conectada (FC) para la realidad local
    num_ftrs = modelo.fc.in_features
    modelo.fc = torch.nn.Sequential(
        torch.nn.Linear(num_ftrs, 256),
        torch.nn.ReLU(),
        torch.nn.Dropout(0.4),
        torch.nn.Linear(256, num_clases)
    )
    return modelo
